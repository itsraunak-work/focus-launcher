## 🧠 Why Focus Launcher?

Your phone is designed to steal your attention. Colorful icons, notification dots, infinite feeds — every element of a default launcher is engineered to keep you scrolling.

**Focus Launcher flips the script.**

No icons. No badges. No distractions. Just your apps, a search bar, and your intentions.

---

## ✨ Features

- 📱 **Pure Minimalism** — Text-only interface, zero visual noise
- ⚡ **Instant Launch** — Open any app in under 2 taps
- 🔍 **Smart Search** — Find and launch apps instantly as you type
- 🔒 **Privacy First** — No internet permission, no tracking, no analytics
- 🔋 **Battery Friendly** — Dark by default, minimal background processes
- 🎨 **Clean UI** — Monochrome design that reduces cognitive load

## 🚀 Run Locally
```bash
# Clone the repo
git clone https://github.com/itsraunak-work/focus-launcher.git

# Open in Android Studio
# File → Open → select the cloned folder

# Run on emulator or physical device
# Click the ▶ Run button in Android Studio
```

**Requirements:**
- Android Studio (latest)
- Android SDK API 26+
- A physical Android device or emulator
